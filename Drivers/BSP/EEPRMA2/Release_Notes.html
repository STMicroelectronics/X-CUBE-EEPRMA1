Purpose

The EEPRMA2 BSP is a SW component for the STM32Cube. This software implements the BSP v2.0 specifications for the X-NUCLEO-EEPRMA2 Expansion Board on STM32 for M24XX/M95XX serial EEPROM. It is built on top of STM32Cube software technology that ease portability across different STM32 micro-controllers
