Purpose

The PGEEZ1 BSP is a SW component for the STM32Cube. This software implements the BSP v2.0 specifications for the X-NUCLEO-PGEEZ1 Expansion Board on STM32 for M95P32 page EEPROM. It is built on top of STM32Cube software technology that ease portability across different STM32 micro-controllers

Here is the list of references to user documents:

  • M95P32 Ultra low-power 32 Mbit Serial SPI Page EEPROM
  • Standard SPI page EEPROM memory expansion board based on M95P32 series for STM32 Nucleo

Update History

Main Changes

Product update

  • Added EEPROMEX_CTRL_HIGH and EEPROMEX_CTRL_LOW functions from component driver
  • Modified PGEEZ1_M95P32_WritePage and PGEEZ1_M95P32_ProgramPage functions to comply with BSP rules
  • Removed PGEEZ1_M95P32_PageProgBuffer function, specific API for buffer mode implemented in user application
  • Comply with BSPv2 rules
