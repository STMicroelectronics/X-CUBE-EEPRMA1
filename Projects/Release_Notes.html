Purpose

The X-CUBE-EEPRMA1 Projects provide examples and applications to aid understanding how to develop an application using the M24xx, M95xx, M95P32, M24256E-F and M24M01E-F EEPROM device. Projects include STM32CubeMX project file (.ioc) for the graphical visualization of the STM32 microcontroller pins, peripherals configuration.

Supported Devices and Boards

  • STM32F401RE devices and STM32F401RE-Nucleo board RevB
  • STM32L053R8 devices and STM32L053R8-Nucleo board RevB
  • STM32G474RE devices and STM32G474RE-Nucleo board RevB
  • STM32G474RE devices and STM32H743ZI-Nucleo board RevB
  • X-NUCLEO-EEICA1
  • X-NUCLEO-EEPRMA2
  • X-NUCLEO-PGEEZ1
  • M24256E-F 256-Kbit serial I2C bus EEPROM with configurable device address
  • M24M01E-F 1 Mbit Serial I2C bus EEPROM with configurable device address and software write protection registers
  • M24C02 2-Kbit serial I2C bus EEPROM
  • M24256 256-Kbit serial I2C bus EEPROM
  • M24M01 1-Mbit serial I2C bus EEPROM
  • M95040 4-Kbit serial SPI bus EEPROM
  • M95256 256-Kbit serial SPI bus EEPROM
  • M95M04 4-Mbit serial SPI bus EEPROM
  • M95P32 Ultra low-power 32 Mbit Serial SPI Page EEPROM
