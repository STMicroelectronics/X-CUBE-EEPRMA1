Purpose

The X-CUBE-EEPRMA1 software package is an expansion for STM32Cube. This software is evaluating M24xx I2C, M95xx and M95P32 SPI EEPROM for reading and writing various size chunk of data. It is built on top of STM32Cube software technology that ease portability across different STM32 microcontrollers.

X-CUBE-EEPRMA1 software features:

  • Complete software to build applications using M24xx or M95x or M95P32 based EEPROM
  • Easy portability across different MCU families thanks to STM32Cube
  • Free user-friendly license terms
  • Example implementation available on the board X-NUCLEO-EEPRMA2, X-NUCLEO-PGEEZ1, STM32 expansion board plugged on top of one NUCLEO-F401RE or NUCLEO-G474RE or NUCLEO-L053R8 or NUCLEO-H743ZI
